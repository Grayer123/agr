# 78. Subsets

[Original Page](https://leetcode.com/problems/subsets/)

Given a set of distinct integers, _nums_, return all possible subsets.

**Note:**  

*   Elements in a subset must be in non-descending order.

For example,  
If **_nums_** = `[1,2,3]`, a solution is:

<pre>[
  [3],
  [1],
  [2],
  [1,2,3],
  [1,3],
  [2,3],
  [1,2],
  []
]
</pre>
