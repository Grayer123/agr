# 226. Invert Binary Tree

[Original Page](https://leetcode.com/problems/invert-binary-tree/)

Invert a binary tree.

<pre>     4
   /   \
  2     7
 / \   / \
1   3 6   9</pre>

to

<pre>     4
   /   \
  7     2
 / \   / \
9   6 3   1</pre>
